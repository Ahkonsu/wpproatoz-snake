== Description ==

Add a fun Snake game to your WordPress site with the `[snake]` shortcode. Forked and updated for modern WordPress and Elementor compatibility.

== Installation ==

1. Upload the `wpproatoz-snake` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add `[snake]` to any post or page (works with Elementor).
